# Proof Artifacts

This directory contains the committed proof artifacts referenced by the README and release notes.

Included files:

- `airline_tweets_benchmark_results.json` — measured benchmark table used for the public results section
- `proof_v7_8b_mv_40iter_final_summary.json` — summary metrics for the featured autonomous run
- `proof_v7_8b_mv_40iter_meta.json` — run metadata for the featured autonomous run

The large working experiment directory remains under `experiments/` locally and is intentionally treated as a workspace artifact rather than curated release content.

The featured charts in `docs/assets/` were copied from the proof run so the public repository can render them directly on GitHub.
